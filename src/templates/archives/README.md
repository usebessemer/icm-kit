# Archives

Retired content lives here. When a file has been superseded and is no longer live, move it into this directory rather than deleting it, so the history stays recoverable without cluttering the working tree.

This directory is walk-ignored by the audit: nothing under `archives/` is read, classified, or checked, so a retired file with an old superseded banner or stale links never produces a finding. That is the point of the home. Move a file here to take it out of the live workspace cleanly; the decision to retire it belongs in [`../decisions/log.md`](../decisions/log.md).
