# Connections

The registry of external systems this install can reach: calendars, mailboxes, drives, repositories, and any other service an agent may act through. It starts empty and fills as the operator wires connections in.

Each entry records what the system is, how it is reached, and what an agent is allowed to do through it. Nothing is assumed to be connected until it is listed here.
